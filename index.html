About

I'm a full-stack developer who builds practical solutions across the entire technology stack. My approach combines robust backend engineering with thoughtful frontend experiences, always focusing on creating software that solves real problems.

On the backend, I specialize in building scalable APIs and distributed systems. I've developed job queue systems in Go with PostgreSQL for reliability, created expense tracking platforms with FastAPI and React, and built modular social media architectures with machine learning capabilities. I'm comfortable working with microservices, message queues, and designing systems that handle concurrent operations gracefully.

I also enjoy frontend development and creating interactive experiences. From building AI-powered games using minimax algorithms to implementing OCR-powered receipt scanning with React and TypeScript, I like bridging the gap between complex backend logic and intuitive user interfaces. I've worked with everything from canvas-based games to ML model integration in the browser.

Beyond traditional web development, I explore diverse areas like roguelike game development in C#, CLI tools for PDF generation, and vector search with FAISS. I'm driven by curiosity and the challenge of learning new technologies to build better solutions.
